## What it does

OG Sorted gives you a complete overview of your site's Open Graph metadata — images, titles, and descriptions — in a single panel inside the Webflow Designer. See at a glance which pages are ready to share, fix the ones that aren't, and apply changes to dozens of pages in a single click.

<CardGrid>
	<Card title="Audit every page" icon="magnifier">
		Per-field status indicators show green, amber, or red for every page. Filter to pages that need attention, search by name, and watch the stats bar track your progress.
	</Card>
	<Card title="Upload images anywhere" icon="add-document">
		Drop zone, file upload, asset library picker, or paste a hosted URL. Live social card preview shows exactly how the page will look when shared.
	</Card>
	<Card title="Bulk apply & remove" icon="document">
		Select any number of pages and apply or clear an OG image in one click, with live progress and retry for failed pages.
	</Card>
	<Card title="CSV import & export" icon="seti:csv">
		Export a full audit, edit in your favourite spreadsheet, and re-import to update images, titles, descriptions, and SEO inheritance in a single pass.
	</Card>
</CardGrid>
